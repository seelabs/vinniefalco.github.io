websocket::stream::read (2 of 2 overloads)

Read a message. 

template<
    class Streambuf>
void
read(
    opcode& op,
    Streambuf& streambuf,
    error_code& ec);

This function is used to read a message from the websocket. The function call will block until the message has been read successfully, or until an error occurs. On success op is set to reflect the message type, binary or text.

Parameters

op

A value to receive the message type. This object must remain valid until the handler is called.

streambuf

A stream buffer to hold the message data. This object must remain valid until the handler is called.

ec

Set to indicate what error occurred, if any.
