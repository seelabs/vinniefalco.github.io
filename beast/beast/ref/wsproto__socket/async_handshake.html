wsproto::socket::async_handshake

Asynchronously send a WebSocket Upgrade request. 

template<
    class HandshakeHandler>
auto
async_handshake(
    boost::string_ref const& host,
    boost::string_ref const& resource,
    HandshakeHandler && h);

This function is used to asynchronously send the WebSocket upgrade HTTP request. This function call always returns immediately.hostThe name of the remote host, required by the HTTP protocol. Copies may be made as needed.resourceThe requesting URI, which may not be empty, required by the HTTP protocol. Copies may be made as needed.hThe handler to be called when the request completes. Copies will be made of the handler as required. The equivalent function signature of the handler must be: voidhandler(error_codeconst&error//resultofoperation); Regardless of whether the asynchronous operation completes immediately or not, the handler will not be invoked from within this function. Invocation of the handler will be performed in a manner equivalent to using boost::asio::io_service::post().
