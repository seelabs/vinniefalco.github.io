http::chunk_encode

Returns a chunk-encoded BufferSequence. 

template<
    class ConstBufferSequence>
implementation-defined
chunk_encode(
    ConstBufferSequence const& buffers,
    bool final_chunk = false);

See: http://www.w3.org/Protocols/rfc2616/rfc2616-sec3.html#sec3.6.1

Parameters

buffers

The input buffer sequence.

final_chunk

true If this should include a final-chunk.

Return Value

A chunk-encoded ConstBufferSequence representing the input.

Requirements

Header: beast/http/chunk_encode.hpp

Convenience header: beast/http.hpp
