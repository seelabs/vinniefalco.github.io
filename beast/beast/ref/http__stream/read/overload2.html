http::stream::read (2 of 2 overloads)

Read a HTTP message from the stream. 

template<
    bool isRequest,
    class Body,
    class Headers>
void
read(
    message< isRequest, Body, Headers >& msg,
    error_code& ec);

This function is used to read a single HTTP message from the stream. The call will block until one of the followign conditions is true:A message has been read. An error occurred. The operation is implemented in terms of zero or more calls to the next layer's read_some function.msgAn object used to store the message. The previous contents of the object will be overwritten.ecSet to indicate what error occurred, if any.
