Requirements

Beast requires:

  • C++11. A minimum of C++11 is needed.
  • Boost. Beast is built on Boost, especially Boost.Asio.
  • OpenSSL. If using TLS/Secure sockets (optional).
Note

Tested compilers: msvc-14+, gcc 5+, clang 3.6+

Most of the library is header-only; however, the HTTP parser used is written in C. To link an application that uses Beast, it is necessary to add a single .cpp file from beast into your project's build script.
